🚀 LEVEL 1 · 배포⏱ 약 12분

인터넷에 내 주소를 만든다

아직 데이터도 그래프도 없어요. 먼저 빈 앱을 인터넷에 올려 내 주소부터 확보합니다. 배포는 오늘 딱 한 번만!

이 레벨에서 얻는 능력🌐 세상 누구나 접속할 수 있는 나만의 웹사이트 주소
  1. 깃허브 로그인 후 오른쪽 위 + ▸ New repository
  2. 이름에 population 입력 · Public 선택 · Add a README file 체크 ▸ Create repository
  3. Add file ▸ Create new file ▸ 파일 이름을 index.html 로
  4. 아래 코드를 복사해서 붙여넣기 👇
📄 index.html (첫 배포용)
  1. 초록색 Commit changes 버튼 클릭
  2. Settings ▸ 왼쪽 Pages ▸ Branch를 main 으로 ▸ Save
  3. 1~2분 뒤, 위쪽에 뜬 내 주소(내아이디.github.io/population)를 새 탭에서 열기
✅ 이렇게 되면 성공! 화면에 "🏘️ 우리 동네 인구 구조" 글자가 인터넷 주소로 보이면 배포 성공. 아직 밋밋해도 정상이에요.
💡 팁 · 반영에 시간이 걸려요 첫 배포는 1~2분(가끔 더) 걸릴 수 있어요. 안 뜨면 잠깐 기다렸다 새로고침!
이후 수정은 보통 수십 초 안에 반영됩니다. 그래도 안 바뀌면 Ctrl+Shift+R(강력 새로고침). 배포는 지금 딱 한 번, 다음부터는 index.html만 고쳐 Commit하면 같은 주소에 자동 반영돼요.
📊 LEVEL 2 · 데이터⏱ 약 12분

우리 동네 인구 데이터를 불러온다

깃허브에 올라가 있는 전국 인구 데이터(CSV) 주소를 AI에게 건네주면, 그 데이터를 불러와 지역별로 보여주는 코드가 알아서 완성됩니다.

이 레벨에서 얻는 능력📊 깃허브에 저장된 공공데이터(CSV)를 내 앱으로 불러오기
  1. 아래 인구 데이터 주소를 복사하세요 (수정할 것 없이 그대로). 새 탭에서 열어보면 진짜 인구 CSV가 보여요! 👇
🔗 인구 데이터 주소 (그대로 복사)
  1. 아래 프롬프트를 복사해 AI(클로드·챗지피티)에 붙여넣고, 맨 끝에 방금 복사한 데이터 주소를 붙여넣어 전송 👇
🤖 AI 프롬프트 (붙여넣고 끝에 데이터 주소 추가)
  1. AI가 준 index.html 코드 전체를 복사 → 깃허브 index.html 편집(연필 ✏️) → 기존 내용 다 지우고 붙여넣기 → Commit changes
  2. 내 주소 탭으로 가서 새로고침(F5)

AI 없이 바로 쓰는 완성본입니다. 그대로 붙여넣어도 되고, 2번째 줄 기본지역만 우리 동네로 바꿔도 돼요. 깃허브 index.html에 통째로 붙여넣으세요.

📄 완성 코드 · 데이터만 (기본지역만 수정 가능)
✅ 이렇게 되면 성공! 검색창에 지역을 입력(예: 서울특별시 종로구)하면 그 지역의 총인구·남자·여자 수가 나타납니다.
💡 왜 신기하냐면 이건 전국 3,900여 개 모든 동네의 진짜 인구예요. 우리 동네, 내 고향, 근무하는 학교 동네까지 다 찾아볼 수 있어요!
📈 LEVEL 3 · 시각화⏱ 약 10분

숫자를 '한눈에 보이는' 그래프로

표 속 숫자만으론 감이 안 오죠. 인구 피라미드로 남녀·연령 구조를 한눈에 보이게 만듭니다.

이 레벨에서 얻는 능력📈 데이터를 그래프로 시각화하기
  1. 레벨 2에서 대화하던 그 AI 창에 아래 프롬프트를 이어서 보내기 👇
🤖 AI 프롬프트 (이어서 대화)
  1. 새 코드를 깃허브 index.html에 통째로 교체 → Commit → 새로고침
📄 완성 코드 · 인구 피라미드
✅ 이렇게 되면 성공! 남자(왼쪽)·여자(오른쪽)가 연령대별로 뻗어나가는 인구 피라미드가 그려집니다.
🤔 이렇게 해보세요 검색창에 다른 동네를 넣어 비교해보세요. 신도시와 시골, 피라미드 모양이 완전히 다를 거예요!
✨ LEVEL 4 · 대시보드⏱ 약 10분

지표 카드까지 갖춘 '진짜 대시보드'

밝은 톤과 지표 카드(고령화율·유소년 비율 등)를 더해 뉴스에 나올 법한 대시보드로 끌어올립니다. 여기서부터가 진짜 바이브코딩이에요.

이 레벨에서 얻는 능력✨ 말 한마디로 데이터 대시보드를 디자인
  1. 같은 AI 창에 아래 프롬프트를 이어서 보내기 → 코드 교체 → Commit → 새로고침 👇
🤖 AI 프롬프트 (밝은 대시보드)
🎨 진짜 바이브코딩은 여기서부터 프롬프트에 한 줄씩 더 시켜보세요 — "내 고향을 기본값으로", "고령화율이 20% 넘으면 빨갛게", "두 지역을 나란히 비교". 말하는 대로 바뀝니다.
📄 완성 코드 · 밝은 대시보드 (테마 전환 포함)
✅ 이렇게 되면 성공! 총인구·고령화율·유소년 지표 카드 + 예쁜 피라미드 + 테마 전환까지 갖춘 대시보드가 완성됩니다.
